### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 论坛
5. 健康常识
6. 疾病预防
7. 网站介绍
8. 体检项目
9. 个人中心
10. 在线咨询


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 在线咨询管理
5. 基础数据管理(健康常识类型管理、科普类型管理、体检项目类型管理)
6. 论坛管理
7. 健康常识管理
8. 疾病预防管理
9. 单页数据管理
10. 体检提醒管理
11. 体检项目管理
